# ToDoApp

# 소개

간단하고 기초적인 기능만 있는 ToDoApp입니다.

- ToDo 할 일 완료 기능
- ToDo 삭제 기능
- ToDo 추가 기능
- ToDos를 LocalStorage에 저장
   - AsyncStorage라이브러리 사용

# 교육

노마드 코더의 "왕초보를 위한 React Native 101"을 보고 만든 ToDoApp입니다.
